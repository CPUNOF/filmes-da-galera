# 🎬 Filmes da Galera (FDG)

Bem-vindo ao **Filmes da Galera**! Este é um projeto desenvolvido exclusivamente para fins de **estudo, prática e aprimoramento** em desenvolvimento web moderno. 

A ideia do projeto nasceu da necessidade de organizar as sessões de cinema de um grupo de amigos, eliminando a eterna dúvida: *"O que vamos assistir hoje?"*. O sistema permite buscar filmes, sugerir para a galera, votar democraticamente e registrar um diário do que já foi visto.

## 🚀 O que o projeto faz?
Para tornar a experiência mais interativa, o site conta com um sistema de **Gamificação**:
* **Acervo Integrado:** Busca de filmes em tempo real (TMDB API).
* **Fila de Sugestões:** Sistema de upvotes (🔥) para decidir o próximo filme.
* **Diário Pessoal:** Espaço privado para registrar e dar notas aos filmes assistidos.
* **Sistema de Recompensas (Cofre):** Usuários ganham pontos e "Ingressos Dourados" (Fura-Fila) ao interagir com o site.
* **Pódio dos Críticos:** Um ranking em tempo real que premia os usuários mais engajados com uma Coroa (👑).

## 🛠️ Tecnologias Utilizadas (Foco de Estudo)
Este projeto foi construído para colocar em prática conceitos de Front-end, Back-end as a Service e consumo de APIs externas:
* **[Next.js](https://nextjs.org/) & [React](https://reactjs.org/)** - Renderização, rotas e componentização.
* **[Tailwind CSS](https://tailwindcss.com/)** - Estilização responsiva e design moderno (Dark Mode/Neon).
* **[Firebase](https://firebase.google.com/)** - Autenticação (Google) e Banco de Dados NoSQL em tempo real (Firestore).
* **[TMDB API](https://www.themoviedb.org/)** - Consumo de dados e imagens do acervo mundial de cinema.
